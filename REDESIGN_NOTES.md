# Hall of Justice Archives 3.0

This release turns the functional curated tracker into the Hall of Justice museum concept:

- DCEU remains the default wing.
- Archive wings now appear immediately after the current mission.
- Cream marble, midnight navy, bronze, and museum-gold replace the cloned dark-dashboard look.
- Catalog entries are presented as exhibit cards with large screen artwork.
- Artwork lookup now searches film and television pages using title, year, format, and universe-specific overrides instead of accepting the first same-name comic page.
- The full local, Firebase, profile, history, analytics, PWA, and GitHub Pages systems remain intact.

Remote artwork is still intentionally used so the repository stays lightweight.
