# Hall of Justice Archives

A cinematic, installable completionist tracker for DC movies, television, animation, serials, specials, and shorts.

## Catalog

Version 1.0 includes **209 source productions** and more than **4,000 trackable films and episodes**, spanning:

- DC Universe (DCU)
- DC Extended Universe (DCEU)
- Arrowverse and connected multiverse series
- DC Animated Universe (DCAU)
- DC Animated Movie Universe (DCAMU)
- Tomorrowverse
- Batman, Superman, and other legacy film continuities
- Elseworlds and standalone productions
- DC Universe Animated Original Movies
- LEGO DC, DC Super Hero Girls, Teen Titans, and other animation
- Golden Age serials and classic television

The default archive is organized by **release order**. Users may switch to a curated chronological view, filter by universe, format, year, or story group, and search by title, character, actor, creator, or continuity.

## Deploy

This repository is configured for GitHub Pages through `.github/workflows/deploy-pages.yml`.

1. Open **Settings → Pages**.
2. Set the source to **GitHub Actions**.
3. Push to `main` or manually run the deployment workflow.

Live site: `https://cinematicarchiveshq.github.io/dcu-completionist-watchlist/`

## Image policy

Poster and backdrop artwork is fetched at runtime from Wikipedia/Wikimedia when available. The project does not host or stream copyrighted movies or episodes.

## Catalog policy

The archive includes screen productions directly based on DC properties and selected DC-published imprints. Unreleased titles are shown separately and do not affect completion percentages. Rumored and canceled projects are excluded.
