# Hall of Justice Archives

A curated, installable DC completion tracker built around distinct archive wings rather than one enormous history-of-DC database.

## Archive wings

- DCEU (default)
- DCU — Chapter One: Gods and Monsters
- Dark Knight Trilogy
- Arrowverse
- DCAMU — New 52
- Tomorrowverse
- LEGO DC
- Animated Elseworlds

Each wing has its own completion total, release-order default, optional chronological order, progress analytics, viewing history, ratings, favorites, notes, profiles, cloud sync, export/restore tools, and PWA installation.

## Deploy

The included GitHub Actions workflow installs dependencies, builds the static Next.js export, validates the artifact, and deploys `out/` to GitHub Pages.
